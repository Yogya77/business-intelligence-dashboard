# Business Intelligence Dashboard

A complete Data Analytics and Business Intelligence Dashboard project for a web development course.

## Features

- KPI cards for revenue, orders, profit, and average order value
- Monthly revenue chart
- Region-wise revenue report
- Category performance report
- Date, region, and category filters
- Export filtered data as CSV
- Data import simulation using sample business data
- Responsive dashboard design

## Tech Used

HTML, CSS, JavaScript, Node.js, JSON data file

## How To Run

1. Install Node.js from https://nodejs.org
2. Open this project folder in VS Code
3. Open the terminal
4. Run: npm start
5. If PowerShell blocks npm, run: npm.cmd start
6. Open: http://localhost:3001

## Repository Description

A business intelligence dashboard for KPI tracking, sales analytics, revenue charts, category reports, and exportable business insights.
